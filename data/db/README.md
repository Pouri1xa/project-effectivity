# Backup data from pgadmin

1. Login to pgadmin
2. Right-click on schema: Backup
3. General:
  1. Format: Plain
  2. Encoding: UTF8
3. Data/Objects:
  1. Pre-data
  2. Data
  3. Post-data
4.  Options:
  1. Use Columns Inserts
  2. Use Insert commands
  3. Use Create database statement
